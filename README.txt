== Description ==
The Payday plugin integrates WooCommerce with your Payday bookkeeping solution.
It also supports OpenPos, allowing handling products to be returned.

After installing the plugin, your admin will have the option to sync Orders with Invoices in Payday automatically.

This plugin is only for companies or individuals who have registered an account with Payday.

Visit our website for more information: http://payday.is

== Installation ==
1. Download the plugin and activate it
2. Login into your Payday.is Account
3. Retrieve your Client ID and Client Secret from Payday
5. Go to WordPress > Payday
6. Fill in the Client ID and Client Secret fields
7. Make sure to Save
